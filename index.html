GINA Puffin Feeder for Android

Build Environment Set-up Instructions:

All Users
  • Install the latest versions of both the Java Runtime Environment (JRE) and the Java Development Kit (JDK).

  • Download either the Eclipse-based ADT or the IntelliJ IDEA-based Android Studio.

    NOTE: If you experience an error when installing Android Studio on Windows, please add a new environment variable titled "JAVA_HOME" that points to the JDK Home directory. If this still does not work, add the full location of your JDK to the PATH variable. MAKE SURE that it comes BEFORE the path leading to "C:/Windows/System32".

  • Navigate to the directory where the ADT or Android Studio was installed (C:\Program Files\Android\android-studio for Android Studio), open "\sdk\tools", and run the file called "android.bat" as admin. Download everything under "Android 4.2.2 (API 17)".

  • Download and install Git and make an account on GitHub if you have not already.

  • Clone the repository in whatever way you see fit.

  • Navigate to the folder where you have just cloned the repository.

Android Studio: (IntelliJ IDEA Community Edition)
  • Open Android Studio and click "Import Project".
  • Navigate to the folder where the repository was cloned and hit "OK".
  • When prompted, choose "Import from External Model" and select "Gradle" in the box beneath and hit "Next".
  • Select "Use Gradle Wrapper" option.
  • Hit finish.
Android Developer Toolkit (ADT): (Eclipse)
  • Navigate to the directory where you downloaded the ADT and extract the files.
  • Go into the folder called "eclipse" and run eclipse.
  • Once it has launched, open the "Help" menu and select "Install New Software...".
  • Enter http://dist.springsource.com/release/TOOLS/gradle into the "Work with:" field and press enter.
  • Select the "Gradle IDE" and "Eclipse Integration Commons" packages and hit next.
  • Accept the license agreements and hit finish.
  • At this point, the ADT will prompt you for a restart. Do so.
  • Open the "File" menu and select "Import...".
  • Select "Gradle Project", found under the "Gradle" category.
  • Select "Browse", navigate to the folder where you cloned the repository, hit "OK", and then click "Build Model".
  • Select both "gina-puffinfeeder-android-viewer" and "PuffinFeeder", then hit finish.
  • To build the project, select the "gina-puffinfeeder-android-viewer" folder, click the "Run" button, and select all of the checkboxes.

    You're done!
